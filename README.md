# Washington D.C. Bike Rental Dashboard

An interactive dashboard analyzing bike rental patterns in Washington D.C. from 2011-2012.

## About

This dashboard explores how weather and temporal factors affect bike rental numbers. The analysis is based on hourly bike rental data from the Capital Bikeshare system.

## Features

The dashboard includes three interactive filters:
- Year selector (2011, 2012, or both years)
- Season filter (spring, summer, fall, winter)
- Day type filter (working days vs non-working days)

Six visualizations show:
- Hourly rental patterns throughout the day
- Rentals by time period (night, morning, afternoon, evening)
- Monthly rental trends
- Seasonal patterns
- Weather impact on rentals
- Temperature correlation with rental demand

## Data Source

Washington D.C. Bike Rental Dataset (2011-2012)
